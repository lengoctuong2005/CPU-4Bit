# Debug Workflow

1. Reproduce issue
2. Collect evidence
3. List all observed errors
4. Build dependency graph
5. Identify root cause
6. Verify root cause
7. Fix root cause only
8. Run affected tests
9. Verify no regression
10. Update failure knowledge if needed
