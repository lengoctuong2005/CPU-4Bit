# Workflow: MCP Docs, Diagrams & Academic Router

Load when: The user requests PDF reading, diagram generation, academic paper research, math solving, UI design inspection, or document conversion.

## Available MCPs in this category:
- `pdf-reader`: Extract text accurately from large PDFs/Datasheets.
- `pdf-export`: Generate professional PDF reports.
- `pandoc`: Convert document formats (Markdown to Word, PDF, LaTeX).
- `kroki`: Generate architecture diagrams, Wavedrom timing diagrams, PlantUML from code.
- `excalidraw`: Generate hand-drawn style mindmaps and block diagrams.
- `wolfram`: Wolfram Alpha computational intelligence (math, physics, logic).
- `arxiv`: Search and download the latest scientific papers.
- `context7`: Search the latest official framework/library documentation.
- `math-mcp`: Solve complex math (integrals, derivatives, matrices) locally.
- `figma`: Connect and read structures, colors, CSS from Figma designs.
- `markdown-converter`: Convert across 34 document formats.
- `markscribe`: Manage notes and sync Markdown/Obsidian structures.

## Usage Guidelines:
1. For modern framework syntax, always check `context7` before writing code to avoid hallucinations.
2. For visual explanations, prefer generating code for `kroki` or `excalidraw`.
3. Figma requires an access token.
