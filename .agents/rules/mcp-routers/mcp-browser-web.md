# Workflow: MCP Browser & Web Scraper Router

Load when: The user requests web automation, web searching, UI testing, website performance auditing, or content scraping.

## Available MCPs in this category:
- `puppeteer` & `playwright`: Virtual browser control, automated clicks, UI testing.
- `tavily`: AI-optimized web search engine.
- `smart-web`: Smart web access and comprehension.
- `fetch`: Raw HTML download from URLs.
- `jina-reader`: Extract clean Markdown from a single URL.
- `firecrawl`: Deep crawl multiple web pages and complex API docs into memory.
- `lighthouse`: SEO, Core Web Vitals, and performance auditing.
- `html-to-markdown`: Convert raw HTML to token-optimized Markdown.

## Usage Guidelines:
1. For quick factual lookups, use `tavily` or `smart-web`.
2. For reading documentation or articles, use `jina-reader`.
3. For deep site crawling, use `firecrawl`.
4. For interactive UI testing (login, clicking buttons), use `puppeteer` or `playwright`.
5. Requires API keys for Tavily and Firecrawl.
