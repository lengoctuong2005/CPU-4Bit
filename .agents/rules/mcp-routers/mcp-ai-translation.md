# Workflow: MCP AI & Translation Router

Load when: The user requests localized AI models, deep translation, or advanced memory management.

## Available MCPs in this category:
- `mem0`: Personalized memory for learning coding styles, habits, and common mistakes.
- `memory`: Global system memory.
- `ollama`: Deploy and control local small AI models (offline inference).
- `deepl`: High-quality, context-aware technical translation.

## Usage Guidelines:
1. Use `deepl` instead of standard LLM translation when precise technical localization is required.
2. Use `ollama` to delegate simple tasks to local models if token/API cost is a concern, or for privacy.
3. Access `mem0` or `memory` to understand user preferences before proposing large architectural changes.
