# Workflow: MCP Media, Social & Utils Router

Load when: The user requests audio/video processing, chart generation, social media posting, or messaging.

## Available MCPs in this category:
- `youtube-transcript`: Extract Youtube subtitles to summarize videos without watching.
- `ffmpeg`: Fast video/audio processing, trimming, format conversion.
- `echarts`: Generate interactive data charts (bar, pie, radar, scatter).
- `spotify`: Control Spotify desktop player.
- `telegram`: Send alerts, messages, or screenshots via Telegram bot.
- `discord`: Interact, summarize tasks, and report bugs to Discord teams.
- `slack`: Slack team integration.
- `rss`: Read news feeds and tech updates automatically.
- `devto`: Publish technical articles/tutorials to Dev.to.
- `gmail`: Read and send emails.
- `hackernews`: Search and summarize tech trends from Hacker News.

## Usage Guidelines:
1. Social media bots (Telegram, Discord, Slack) require Bot Tokens.
2. Ensure you have the exact Chat ID or Channel ID before sending messages.
3. For data visualization, format JSON correctly for `echarts`.
