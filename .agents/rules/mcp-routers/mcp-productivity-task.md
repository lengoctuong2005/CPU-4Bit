# Workflow: MCP Productivity & Task Management Router

Load when: The user requests interaction with task boards, calendars, personal notes, cloud storage, or study tools.

## Available MCPs in this category:
- `notion`: Sync data and store knowledge in Notion.
- `linear`: Manage tasks and project progress (Agile/Kanban).
- `calendar`: View and manage Google Calendar events.
- `gdrive`: Retrieve and read files directly from Google Drive.
- `anki`: Automatically generate study flashcards.
- `google-maps`: Get map coordinates and traffic distance.
- `google-sheets`: Read/write/manipulate Google Sheets.
- `trello`: Manage visual Kanban boards.
- `todoist`: Manage daily to-do lists.
- `airtable`: Manage visual database projects.

## Usage Guidelines:
1. Most of these require API tokens or OAuth credentials.
2. Confirm with the user which specific page/board ID they want to modify before writing data.
